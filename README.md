# veganbite_restaurant
Responsive vegan restaurant website built with React, Bootstrap, custom CSS, and JavaScript. Includes menu, gallery slider, contact form, Google Map, and dynamic shopping cart. The project rebuilds the original restaurant website using React, Bootstrap, JavaScript, HTML, and custom CSS.

## Project Description

This website represents a modern vegan restaurant with a clean plant-based theme. It includes a homepage, menu section, gallery slider, About Us section, Contact section, Google Map, and a working shopping cart.

Users can browse vegan menu items, add items to the cart, view item quantities and prices, remove items, clear the cart, and see the total update dynamically.

## Features

- Responsive navigation bar
- Mobile-friendly hamburger menu
- Hero landing section
- Vegan menu with category filters
- Dynamic shopping cart
- Gallery image slider
- About Us section
- Contact form
- Embedded Google Map
- Footer with social links and business hours
- Responsive layout for desktop, tablet, and mobile screens

## Technologies Used

- React
- Bootstrap
- HTML
- CSS
- JavaScript

## Purpose
The goal was to rebuild the restaurant website (https://veganbitee.netlify.app) using React and style it with a CSS framework while making the website as responsive as possible.
